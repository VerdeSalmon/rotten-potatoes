# rotten-potatoes

## Description:
[Rotten Potatoes](https://rotten-potatoes-dbc.herokuapp.com) -- A website to view and rate movies a la Rotten Tomatoes. Built on Rails, deployed on Heroku. API credit [The Movie DB](https://www.themoviedb.org/).

## Tech:
Ruby on Rails
PostgreSQL db
Bootstrap
